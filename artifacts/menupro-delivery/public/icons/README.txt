Ces icônes sont des placeholders.
Pour générer de vraies icônes PNG orange avec la lettre M :
  npm install -g @squoosh/cli  # ou utiliser un outil en ligne

Ou utiliser ce SVG comme source :
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 192 192">
  <rect width="192" height="192" rx="40" fill="#f97316"/>
  <text x="96" y="130" font-family="Arial" font-size="110" font-weight="bold" fill="white" text-anchor="middle">M</text>
</svg>
